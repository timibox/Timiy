Hello 我的网站诞生了，它使用的是Hexo开发，部署再Github、使用github action 持续部署、使用 BRIEF 极简化轻社交主题作为前端渲染 后端使用的是`HexoPlusPlus后台`

### 快速安装：

安装hexo

``` bash
$ npm install hexo-cli -g
```

初始化博客

``` bash
$ hexo init blog
```

进入博客目录

``` bash
$ cd blog
```

执行安装

``` bash
$ npm install
```

运行Hexo
``` bash
$ hexo server
```
